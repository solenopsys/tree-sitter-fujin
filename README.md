# Tree-sitter Fujin

Parser grammar for the Fujin language (actors, emit, lightweight FJX tags). See the full language docs and spec at https://github.com/solenopsys/fujin.

## Usage

```
pnpm install
pnpm build   # runs `tree-sitter generate`
```

Outputs are in `src/` and `build/` per tree-sitter defaults.***
